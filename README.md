# Dijkstra's Algorithm Shortest Path Finder

Консольное C++ приложение для поиска кратчайшего пути в ориентированном/неориентированном графе с использованием классического алгоритма Дейкстры. Проект демонстрирует применение объектно-ориентированного программирования (ООП), работу с динамической памятью и теорию графов.

# Стек технологий
* **Язык:** C++
* **Архитектура:** Объектно-ориентированная (класс `Graph`)
* **Фреймворк:** Qt Core (`QFile`, `QTextStream`, `QStack` для алгоритма восстановления пути)

# Основной функционал
1. **Чтение графа из файла:** Загрузка количества вершин, ребер и весов из текстового файла (`graph.txt`) с обработкой ошибок открытия файла.
2. **Матрица смежности:** Хранение графа реализовано в виде динамического двумерного массива (`int**`), что обеспечивает прямой доступ  к весу любого ребра.
3. **Алгоритм Дейкстры:** Вычисление минимальных расстояний от стартовой вершины до всех остальных.
4. **Восстановление пути:** Использование стека (LIFO) для отслеживания массива предков (`parent`) и пошагового восстановления оптимального маршрута от начальной до целевой точки.
5. **Memory Management:** Строгий контроль утечек памяти — ручное освобождение двумерных массивов и указателей в деструкторе класса.

## 🗄️ Формат входного файла (graph.txt)
Первая строка содержит количество вершин `N` и количество ребер `M`.
Следующие `M` строк описывают ребра в формате: `[Откуда] [Куда] [Вес]`.

Пример:
```text
6 8
1 2 5
1 3 2
3 2 1
...
